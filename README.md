# universal-usage-bar
it can be useful for games to know what are ram cpu and disk usage are
and how to use: you need to download python 3.13.9 or higher
and you need to install psutil and GPUtil and setuptools for it to work and how to install psutil and GPUtil and setuptools is pip install psutil and for gputil it is pip install gputil and pip install setuptools 
make sure you put the commands in windows powershell
